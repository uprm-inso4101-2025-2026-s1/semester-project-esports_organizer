= Statechart Diagram for Roles and Permissions
This statechart provides examples of the permissions each role has access to in the Esports Organizer.

== Permissions List

=== Universal Permissions
These permissions apply to all roles:

* viewTournaments
* createCommunities
* canCreatePrivateTournaments
* canCreatePublicTournaments
* joinTournament
* joinCommunities

=== Player Role
Role-specific permissions for Player:

* canEditUserProfile
* requestToJoinTeam
* editUserEvent
* createUserEvent
* removeUserEvent

=== Team Manager Role
Role-specific permissions for Team Manager:

* joinTournamentwithTeam
* editTeamProfile
* editTeamRoster
* invitePlayerstoTeam
* assignTeamClasses
* addPlayerToRoster
* removePlayerFromRoster
* editTeamEvent
* createTeamEvent
* removeTeamEvent

=== Admin Role
Role-specific permissions for Admin:

* canBanUser
* canSendWorldwideNotifications
* removeEvents
* editTeamProfile
* editTeamRoster
* invitePlayerstoTeam
* assignTeamClasses
* addPlayerToRoster
* removePlayerFromRoster
* editTeamEvent
* createTeamEvent
* removeTeamEvent
* canEditUserProfile
* requestToJoinTeam
* editUserEvent
* createUserEvent
* removeUserEvent
