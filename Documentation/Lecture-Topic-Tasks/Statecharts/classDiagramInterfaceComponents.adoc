= Navigation Interface Class Diagram Documentation
Esports Organizer 


==  Summary
This document describes the class structure used for the navigation interface of the Esports Organizer web application.
It includes the Navbar, its UI components, and the main pages that users can navigate to. The goal is to provide a clear,
concise architectural view of how navigation elements relate and how they support the user flow throughout the system.

== Component Descriptions

=== Navbar
The main navigation component containing all primary navigation links and UI-level interactive elements.
It manages page selection, highlighting, and routing behavior.

=== NavItem
Represents one clickable navigation option. Stores its text, icon, route, and active/inactive state.

=== HomePage
The landing page for logged-in users. Displays recommended events and communities.

=== EventsPage
Allows searching, browsing, and filtering tournaments/events.

=== TeamsPage
Displays teams, their stats, and a search bar for filtering.

=== CommunityPage
Shows user-followed communities and recommended new ones.

=== HelpPage
Contains FAQs and search functionality for help-related topics.

=== AdminPage
Provides admin-only tools and user management functions.

=== UserMenu
User dropdown panel with profile options and preferences.

=== NotificationIcon
Displays unread notification count and opens/closes the notifications panel.
